# Wild! Wild! Wild! Pay Tables

Pay tables transcribed from in-game screenshots. Each game shows the base pay (1-coin column)
and the max bet pay (5-coin column, which includes WWW feature boosts).

The WWW feature requires a 10-coin bet: 5 coins base + 5 coins feature fee. Wins pay based on
the 5-coin bet. The max bet column reflects the boosted payouts when the feature is active.

---

variant and even by pay table within a variant. Data sourced from
